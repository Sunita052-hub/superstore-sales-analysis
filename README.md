# Superstore Sales Analysis

## Project Overview
This project analyzes Superstore sales data using Python to identify sales trends, profit performance, and customer insights.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib

## Project Objectives
- Clean and preprocess the dataset
- Perform exploratory data analysis (EDA)
- Analyze sales and profit trends
- Identify top-performing categories and regions
- Visualize key business insights

## Key Insights

- **Total Revenue:** $2,297,201 | **Total Profit:** $286,397 | **Profit Margin:** 12.5%
- **5,009 orders** from **793 customers** analyzed across 4 years (2014–2017)
- **Technology** was the top-selling category ($836K), ahead of Furniture ($742K) and Office Supplies ($719K)
- **West region** was the most profitable ($108K profit), while **Central** lagged furthest behind ($40K)
- **Copiers** was the most profitable sub-category; **Tables** consistently lost money
- **High discounts (40%+) strongly correlate with negative profit** — most loss-making orders cluster above the 40% discount mark, while low/no-discount orders stay reliably profitable
- Monthly sales show a clear **upward trend from 2014 to 2017**, with recurring year-end spikes suggesting seasonal demand


## Business Recommendation

Reduce or eliminate discounts above 30–40%, especially on the Tables sub-category, since deep discounting is directly linked to loss-making orders. Reallocating marketing/inventory focus toward Copiers and the West region — the strongest profit drivers — could improve overall margin without cutting revenue.
